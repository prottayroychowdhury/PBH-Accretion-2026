# pbh17

`pbh17` is my implementation of a base package for the primordial black hole accretion model of Ali-Haïmoud and Kamionkowski (2017).

It defines the background cosmology and recombination history, Bondi accretion quantities, Compton drag and cooling, temperature profile in the collisional and photoionization limits, radiative efficiency, and PBH luminosity as presented in the paper.

The package is intended to provide the original 2017 model as a common baseline. New corrections and extensions will be implemented outside `pbh17` so they can be compared directly against the published results.

Reference: [Cosmic microwave background limits on accreting primordial black holes](https://arxiv.org/abs/1612.05644).
